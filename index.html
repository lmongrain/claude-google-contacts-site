Claude Google Contacts Connector

This application allows users to securely connect their Google Contacts to Claude so they can quickly find and use contact information when composing emails.

Purpose of the Application

The purpose of this application is to help users retrieve contact details from their own Google Contacts account while using Claude.

How the Application Works

When a user asks Claude to find a contact, this application securely queries the user’s Google Contacts account and returns relevant details such as contact name, email address, or phone number.

Google User Data Accessed

This application accesses Google Contacts data, including names, email addresses, and phone numbers.

How Google User Data Is Used

Data is used only to fulfill user-initiated requests, such as finding a contact when composing an email.

Important Limitations

This application operates in read-only mode. It does not create, modify, or delete contacts.

No data is shared with third parties and no contact data is stored permanently.
