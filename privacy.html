Privacy Policy

This application allows users to connect their Google Contacts to Claude in order to retrieve contact information when requested by the user.

Data Access

The application accesses Google Contacts data only when explicitly requested by the user.

Data Usage

Contact data is used solely to fulfill user requests, such as finding a contact when composing an email.

Data Storage

No contact data is stored permanently on our servers.

Data Sharing

No data is shared with third parties.

User Control

All access to Google Contacts is initiated by the user and can be revoked at any time.

Security

The application uses secure OAuth authentication provided by Google to access user data.

Contact

If you have any questions about this Privacy Policy, you can contact us at admin@evirum.com.
