Privacy Policy

This application allows users to connect their Google Contacts to Claude in order to retrieve contact information when explicitly requested by the user.

Data Access

The application accesses Google Contacts data, including names, email addresses, and phone numbers, only when explicitly requested by the user.

Data Usage

Contact data is used solely to fulfill user requests, such as finding a contact when composing an email. The data is processed in real time and is not used for any other purpose.

Data Sharing

No Google user data is shared with third parties.

Data Storage

The application does not permanently store Google Contacts data. Any data retrieved is used temporarily to fulfill the user's request.

Data Retention and Deletion

No contact data is retained after the request is completed. Since no data is stored, there is no need for deletion requests. Users can revoke access at any time via their Google account permissions.

Data Protection

The application uses secure OAuth authentication provided by Google. All communications are encrypted using HTTPS, and access tokens are securely handled and validated.

User Control

All access to Google Contacts is initiated by the user and can be revoked at any time.

Compliance

This application complies with the Google API Services User Data Policy.
